# FreelanceFlow

Mobile-first app for freelancers to track invoices, send payment reminders, and visualize cash flow.

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- Billing UI
- Supabase Auth
- Supabase Database
- Edge Functions
- CDN / Cache Layer
- Payment integration

## Getting Started

```bash
npm install
cp .env.example .env
# Fill in your Supabase credentials in .env
npm run dev
```

## Environment Variables

| Variable | Description |
|----------|-------------|
| VITE_SUPABASE_URL | Your Supabase project URL |
| VITE_SUPABASE_ANON_KEY | Your Supabase anon/public key |

## Features

- Invoice generator
- Payment reminders
- Cash flow dashboard
- Client portal
- Tax export
